
## 指纹特征提取系统

![](GUI.png)

## 测试环境

- Python == 3.8
- PySimpleGUI == 4.60.5
- opencv-contrib-python == 4.8.0.76
- Numpy == 1.23.1
- Scipy == 1.9.0

## 使用方法

1. 运行`GUI.py`文件
2. 点击`选择图片`按钮，选择一张图片
3. 点击`一键处理`按钮，等待特征提取完成
4. 点击`退出`按钮，退出程序
